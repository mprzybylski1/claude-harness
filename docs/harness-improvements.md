# Harness Improvements Log

Track improvements by scope so the pivot to targeted outreach doesn't require reverse-engineering what was built for whom.

**Generalizable:** improvements that will transfer to engineering leaders at 10–50 person teams (higher-trust, longer cycle, higher stakes).  
**Unknown:** needs more evidence before classifying.

When pivoting to the target buyer, audit the Upwork-scoped entries first — these are the ones most likely to need rework.
